improvement: **Render-benchmark budget ratchet (`bench-render --check`)**: `zig build bench-render -Doptimize=ReleaseFast -- --check tools/bench-render-budgets.txt` runs the deterministic scenario suite three times, takes the median end-to-end p50 per scenario, and fails against committed per-scenario budgets — calibrated with ~1.3x+ headroom so it trips on order-of-magnitude regressions (a reintroduced quadratic planner scan, an extra emission per input), never machine noise. Every scenario must have a budget and every budget must name a scenario, so a rename cannot silently un-gate; the check refuses to run outside ReleaseFast.
- **Gate wiring**: `scripts/gate.sh` runs the ratchet by default on macOS — fast tier for framework diffs (engine perf cannot regress from an example-only diff), full tier always — with `NATIVE_SDK_SKIP_BENCH_CHECK=1` as the busy-box escape hatch; unlike the opt-in `--perf` live harness, the headless in-process benchmark is load-tolerant enough to gate unconditionally.
