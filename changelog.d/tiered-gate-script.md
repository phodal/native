improvement: **Tiered local gate (`scripts/gate.sh`)**: `gate.sh fast [base-ref]` runs the root tests + manifest validation plus the suites for only the examples your diff actually touches (framework changes fan out to all example suites; a docs-only diff runs only the docs check), while `gate.sh full` runs everything CI-shaped that works locally — all example suites, the four macOS GPU smokes, the mobile layout checks, a markup check over every example `.zml`, and the path-gated docs check (`--all` forces it).
