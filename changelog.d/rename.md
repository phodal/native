improvement: **zero-native is now the Native SDK**: the toolkit, CLI, and package ecosystem are renamed. The CLI binary is `native` (`native init`, `native markup check`, `native automate ...`, `native skills get`); the Zig module and build-helper import is `native_sdk` (`@import("native_sdk")`, `native_sdk.addApp`, `native_sdk.addMobileLib`, `.name = .native_sdk` in `build.zig.zon`); the embed C ABI prefix is `native_sdk_*` (was `zero_native_*`, including the platform host symbols `native_sdk_appkit_*`/`_gtk_*`/`_windows_*`); the builtin bridge command namespace is `native-sdk.*` (e.g. `native-sdk.dialog.saveFile`); the automation directory is `.zig-cache/native-sdk-automation/`; and the npm CLI package is `@native-sdk/cli` with release assets named `native-sdk-<platform>`.
