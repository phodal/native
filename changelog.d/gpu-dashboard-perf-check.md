feature: **Percentile GPU perf check (`zig build test-gpu-dashboard-perf`)**: launches the automation gpu-dashboard N times (`ZN_PERF_LAUNCHES`, default 5) asserting p90 first-frame latency under `ZN_PERF_BUDGET_MS` (default 300 ms), then drives M steady-state "Auto refresh" widget clicks (`ZN_PERF_INTERACTIONS`, default 5) asserting p90 input-to-present latency under `ZN_PERF_INPUT_BUDGET_MS` (default 100 ms) — every sample printed, nearest-rank percentiles, so perf regressions are caught by a distribution instead of the smoke's single load-tolerant sample. Runs via `scripts/gate.sh full --perf` locally and a dedicated macos-14 CI job; deliberately excluded from `zig build test` and the fast gate tier.
