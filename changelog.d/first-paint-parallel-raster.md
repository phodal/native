improvement: **First paint rasterizes across cores**: a full-surface packet pass now collects every raster-cache miss up front and rasterizes them concurrently (the pure build half runs per-thread; cache stores and the z-order blit stay serialized), cutting the first full pass from the sum of its rasters to roughly its longest one. Output is byte-identical to serial fills by construction.
- **Image draws ride the raster cache on the CPU path too**: a drawn image (fit, sampling, corner mask) is a pure function of the command plus the registered pixels, so `draw_image` without a transform now caches its scaled output like text and shadows do — the per-present resample that dominated image-heavy first frames becomes a one-time cost; cached image rasters drop whenever an image upload/evict action lands.
- Measured warm launch-to-glass (interleaved before/after batches, medians): soundboard 150→120 ms, system-monitor 116→107 ms, gpu-dashboard 114→105 ms, deck 102→95 ms.
