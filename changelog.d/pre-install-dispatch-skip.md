perf: **Pre-install dispatches no longer rebuild the view**: appearance and chrome events land before the first frame on every launch, and each used to trigger a full view build at a default surface size whose output was discarded — the installing rebuild renders the accumulated model state anyway. `dispatch` now applies the message and defers rendering until the view is installed.
- **Launch lap channel extended**: `NATIVE_SDK_WINDOW_TIMING=1` now also stamps `init_fx_done`, `first_reconcile_begin`, `first_display_list_emitted`, and `first_plan_begin`/`first_plan_done`, splitting the startup frame into boot-effect, build, reconcile+emit, plan, and encode segments; `NATIVE_SDK_GPU_DRAW_TRACE_KINDS=1` adds per-command attribution for slow host rasters.
