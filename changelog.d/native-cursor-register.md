improvement: **Native cursor register**: the engine now follows native platform convention for the pointer cursor instead of the web's — every control (buttons, toggles, checkboxes, radios, switches, selects, menu items, tabs/segmented controls, list rows, accordion headers, sliders) keeps the arrow, and the pointing hand is reserved for true hyperlinks (link spans and `role=link` targets). I-beam over editable text and resize arrows over split dividers/resizable edges are unchanged, and the docs live previews inherit the register through the existing cursor mirror.
