feature: **Open-select keymap**: a focused select/combobox opens from Enter, Space, or the arrow keys, arrows walk into the mounted anchored menu (entering at the selected option), Home/End jump to the edges, Enter commits, and Escape dismisses — with keyboard focus returning to the trigger on commit and dismissal alike, all engine-owned and automation-drivable via `widget-key`.
- **Trailing icon slot**: `icon-placement="trailing"` (markup) / `ElementOptions.icon_placement` (Zig) puts a button's icon after its label — the next-page chevron — and the built-in icon set gains `ellipsis`.
