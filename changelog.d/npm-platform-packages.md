feature: **Publish-ready npm distribution**: `@native-sdk/cli` now installs with zero scripts — the `native` binary ships in per-platform packages (`@native-sdk/cli-<platform>`, eight targets) selected as optional dependencies, and the SDK source ships inside the main package so `native init && native dev` work offline right after install.
- **SDK resolution for npm installs**: the CLI derives the SDK root from its own location in the platform-split layout (and the wrapper passes `NATIVE_SDK_PATH`), so apps scaffolded by an npm-installed `native` build without configuration.
- **One-command cross-builds**: `zig build cli` builds only the CLI, and `packages/native-sdk/scripts/build-binaries.sh` stages all eight platform binaries plus checksummed release assets; the release workflow publishes the whole package set on a version bump, gated on the `NPM_TOKEN` secret.
